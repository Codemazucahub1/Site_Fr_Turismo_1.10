# 🌍 FR Turismo Executivo

Site institucional desenvolvido na Sprint 3 da faculdade com o objetivo de melhorar a presença digital da empresa FR Turismo Executivo, especializada em viagens turísticas agendadas.

O projeto foi desenvolvido utilizando HTML e CSS, com foco em apresentar os serviços da empresa de forma clara, moderna e acessível.

---

## 🎯 Objetivo

O projeto teve como tema o desenvolvimento de um site para auxiliar microempreendedores a aumentar sua visibilidade no mercado digital.

A empresa escolhida foi a FR Turismo Executivo. A partir disso, foi realizada a análise das necessidades do cliente, entendendo seus objetivos e os elementos que deveriam compor o site.

O desenvolvimento foi realizado em equipe de 3 pessoas, com foco na criação de uma solução funcional e bem estruturada.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
